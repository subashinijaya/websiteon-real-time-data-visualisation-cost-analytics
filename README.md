# Websiteon: Distributed Data Visualisation with Cost Analytics

## Project Overview
Websiteon is a distributed data visualisation platform developed as part of the MDA692 capstone project. The main aim of this project is to design and implement a system that provides real-time data analytics along with cost visibility for data processing.

This project is based on my academic work and focuses on applying modern data analytics, data engineering, and visualisation techniques to solve real-world problems.

---

## Objectives
- Develop a scalable distributed data analytics system
- Provide real-time insights through interactive dashboards
- Integrate cost monitoring into data processing workflows
- Support data-driven decision-making
- Demonstrate practical application of data analytics concepts

---

## System Architecture
The system follows a distributed architecture with multiple layers:

- **Data Ingestion Layer** – collects data from different sources  
- **Data Processing Layer** – cleans and transforms raw data  
- **Storage Layer** – stores structured and processed data  
- **Visualisation Layer** – presents insights using dashboards
